En informatique et en mathématiques discrètes, une inversion est un couple de positions dans une suite telles que les éléments situés à ces positions ne suivent pas leur ordre naturel. Ainsi, si nous utilisons l'ordre croissant pour trier une suite de nombres, une inversion se produit quand, dans cette suite, des nombres plus grands apparaissent avant des nombres plus petits.

Observons par exemple cette suite de nombres : (1, 2, 5, 3, 4, 7, 6) ; nous pouvons voir qu'il y a ici trois inversions :
- 5 et 3 ; - 5 et 4 ; - 7 et 6.

On vous donne une suite de nombres différents et vous devez compter le nombre d'inversions dans cette suite.

Entrée : Une suite de nombres, en tant que tuple d'entiers.

Sortie : Le nombre d'inversions, en tant qu'entier.

Exemple : 

count_inversion((1, 2, 5, 3, 4, 7, 6)) == 3
count_inversion((0, 1, 2, 3)) == 0

À quoi cela sert : Dans cette mission, vous allez pouvoir utiliser des boucles imbriquées, si bien sûr, vous n'utilisez pas d'algorithme plus avancé.

Précondition : 2 < len(sequence) < 200
len(sequence) == len(set(sequence))
all(-100 < x < 100 for x in sequence)
